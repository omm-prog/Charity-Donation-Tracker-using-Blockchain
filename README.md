## 🚀 Quick Start

### 📋 Prerequisites

<table>
<tr>
<td width="50%">

**💻 Development Environment**
- ✅ Node.js (v14.0.0+)
- ✅ npm or yarn package manager
- ✅ Git version control

</td>
<td width="50%">

**🔐 Blockchain Setup**
- ✅ MetaMask wallet extension
- ✅ Ganache for local development
- ✅ Basic knowledge of Ethereum

</td>
</tr>
</table>

### ⚡ Installation

<details>
<summary><b>🔧 Step 1: Clone & Setup</b></summary>

```bash
# Clone the repository
git clone https://github.com/your-username/DonationTrack-Blockchain-Crowdfunding.git
cd DonationTrack-Blockchain-Crowdfunding

# Install frontend dependencies
cd crowdfunding-ui
npm install

# Install smart contract dependencies
cd ../smart-contracts
npm install
```

</details>

<details>
<summary><b>⚙️ Step 2: Environment Configuration</b></summary>

```bash
# Create environment file
cd crowdfunding-ui
cp .env.example .env
```

**Configure your `.env` file:**
```env
# Firebase Configuration
REACT_APP_FIREBASE_API_KEY=your_firebase_api_key
REACT_APP_FIREBASE_AUTH_DOMAIN=your_firebase_auth_domain
REACT_APP_FIREBASE_PROJECT_ID=your_firebase_project_id
REACT_APP_FIREBASE_STORAGE_BUCKET=your_firebase_storage_bucket
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
REACT_APP_FIREBASE_APP_ID=your_app_id

# Blockchain Configuration
REACT_APP_CONTRACT_ADDRESS=your_deployed_contract_address
```

</details>

<details>
<summary><b>⛓️ Step 3: Blockchain Setup</b></summary>

1. **Launch Ganache**
   - Create new workspace
   - Note RPC Server URL: `http://127.0.0.1:7545`

2. **Configure MetaMask**
   - Add Ganache network (Network ID: 5777)
   - Import Ganache account using private key

3. **Deploy Smart Contracts**
   ```bash
   cd smart-contracts
   npx hardhat run scripts/deploy.js --network ganache
   ```
   
   ⚠️ **Important:** Copy the deployed contract address to your `.env` file

</details>

<details>
<summary><b>🚀 Step 4: Launch Application</b></summary>

```bash
cd crowdfunding-ui
npm start
```

🎉 **Success!** Visit [http://localhost:3000](http://localhost:3000)

</details>

### 🏢 **For NGOs**

<details>
<summary><b>🔐 Registration & Profile Setup</b></summary>

1. **Create Account**
   - Click "Sign Up" → Select "NGO"
   - Complete registration form
   - Verify email address

2. **Profile Setup**
   - Add organization details
   - Upload verification documents
   - Set up profile image and description

</details>

<details>
<summary><b>📋 Campaign Management</b></summary>

1. **Create Campaign**
   - Access NGO dashboard
   - Click "Create New Campaign"
   - Fill campaign details (name, goal, duration)
   - Submit for approval

2. **Track Progress**
   - Monitor donations in real-time
   - View donor analytics
   - Send updates to supporters

</details>

<details>
<summary><b>💰 Fund Management</b></summary>

1. **Withdraw Funds**
   - Submit proof of fund usage
   - Wait for verification
   - Withdraw approved funds

2. **Impact Reporting**
   - Upload receipts and documentation
   - Share impact stories
   - Build donor trust

</details>

<details>
